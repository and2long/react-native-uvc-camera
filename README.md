# react-native-uvc-camera

UVC Camera for RN

## Installation

```sh
npm install react-native-uvc-camera
```

## Usage

```js
import { UvcCameraView } from "react-native-uvc-camera";

// ...

<UvcCameraView color="tomato" />
```
